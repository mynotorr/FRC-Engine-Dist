# NOTICE UTILISATEUR — FRC Engine v2.1
==================================================================================

## Présentation

**FRC Engine v2.1** est une suite d’optimisation Windows orientée **gaming compétitif**, conçue pour améliorer la **réactivité**, réduire l’input lag, stabiliser les frametimes et maintenir un système plus propre pendant les sessions de jeu.

Le logiciel agit à plusieurs niveaux :

- optimisation Windows
- nettoyage mémoire
- maintenance temps réel
- surveillance du contexte de jeu
- gestion du focus application / jeu
- overlay de monitoring
- gestion intelligente des modes de performance
- boost global simplifié avec **ALL IN ONE BOOST**

FRC Engine n’est pas un simple “nettoyeur”.  
Il agit aussi sur des paramètres système avancés afin d’obtenir un comportement plus réactif, plus stable et plus cohérent pendant les sessions gaming.

---

## Important avant utilisation

### 1. Exécution en administrateur

FRC Engine doit être lancé avec les **droits administrateur** pour appliquer correctement certaines optimisations système et accéder aux fonctions avancées.

### 2. Utilisation raisonnée

Certaines optimisations système profondes n’ont **pas besoin d’être relancées tous les jours**.

Les fonctions de maintenance live, elles, peuvent rester actives pendant les sessions de jeu.

### 3. Redémarrage

Certaines optimisations nécessitent un **redémarrage du PC** pour être appliquées complètement.

Après une optimisation Windows, le bouton peut afficher **REDÉMARRER** afin d’indiquer qu’un redémarrage est recommandé.

Après redémarrage du PC, FRC Engine revient automatiquement à son état normal avec le bouton **optimisation windows**.

### 4. Modules live indépendants

Les modules live peuvent être utilisés **sans lancer l’optimisation Windows complète**.

Vous pouvez donc utiliser directement :

- **APP FOCUS**
- **GAME FOCUS**
- **LOW LATENCY**
- **FPS BOOST**
- **ALL IN ONE BOOST**
- **AUTO-CLEAN**
- **OVERLAY**

sans forcément modifier Windows en profondeur.

### 5. Compatibilité

FRC Engine est conçu pour Windows et cible principalement un usage :

- gaming compétitif
- recherche de faible latence
- optimisation manuelle / assistée du système
- maintien d’un environnement plus propre pendant les sessions

---

## Philosophie de fonctionnement

FRC Engine v2.1 repose sur deux grands principes.

### A. Optimiser le système

Le moteur applique des optimisations afin de rendre Windows plus réactif, plus propre et plus cohérent pour le jeu.

### B. Maintenir la performance en temps réel

Pendant que vous jouez, le moteur continue de surveiller et d’ajuster certains comportements afin d’éviter que le système soit perturbé par des tâches inutiles ou des changements de contexte mal gérés.

La version 2.1 ajoute aussi une logique plus simple avec **ALL IN ONE BOOST**, pensée pour activer rapidement un environnement orienté performance sans devoir gérer chaque module un par un.

## APP FOCUS

Mode de focalisation application.

**APP FOCUS** est conçu pour renforcer la priorité et la cohérence de l’application active.

Son rôle est d’aider Windows à mieux se concentrer sur ce qui est réellement utilisé au premier plan.

### Ce que fait ce mode

- surveille l’application active
- renforce le focus sur la fenêtre utilisée
- améliore la cohérence du comportement système
- limite certaines perturbations en arrière-plan
- accompagne les jeux et les applications importantes

### Important

APP FOCUS ne remplace pas GAME FOCUS.

Il agit comme une couche complémentaire qui améliore le comportement global du moteur, même lorsque le contexte n’est pas strictement identifié comme un jeu.

---

## GAME FOCUS

Mode de focalisation jeu.

Le rôle de ce module est de détecter le contexte de jeu actif et d’orienter le comportement du moteur pour favoriser le jeu au premier plan.

### Ce que fait ce mode

- surveille le contexte actif
- détecte le jeu au premier plan
- renforce l’attention portée au jeu détecté
- améliore la cohérence des optimisations live quand vous êtes réellement en jeu
- aide à mieux gérer les transitions jeu / bureau
- évite autant que possible qu’un ancien état reste “collé” après retour bureau ou Alt+Tab

### Important

GAME FOCUS est fait pour être **réactif**, mais aussi **stable**, afin d’éviter les changements erratiques.

## Overview

**FRC Engine v2.1** is a Windows optimization suite designed for **competitive gaming**, built to improve **responsiveness**, reduce input lag, stabilize frametimes, and keep the system cleaner during gaming sessions.

The software works on several levels:

- Windows optimization
- memory cleanup
- real-time maintenance
- game context monitoring
- application / game focus handling
- monitoring overlay
- intelligent performance mode management
- simplified global boost with **ALL IN ONE BOOST**

FRC Engine is not just a simple “cleaner.”  
It also works on advanced system settings to deliver a more responsive, more stable, and more consistent in-game behavior.

---

## Important Before Use

### 1. Run as Administrator

FRC Engine must be launched with **administrator privileges** in order to properly apply certain system optimizations and access advanced functions.

### 2. Use It Properly

Some deep system optimizations do **not need to be reapplied every day**.

Live maintenance functions, however, can remain active during gaming sessions.

### 3. Restart

Some optimizations require a **PC restart** to be fully applied.

After Windows optimization, the button may display **RESTART** to indicate that a restart is recommended.

After restarting the PC, FRC Engine automatically returns to its normal state with the **optimisation windows** button.

### 4. Independent Live Modules

FRC Engine live modules can be used **without running the full Windows optimization**.

You can directly use:

- **APP FOCUS**
- **GAME FOCUS**
- **LOW LATENCY**
- **FPS BOOST**
- **ALL IN ONE BOOST**
- **AUTO-CLEAN**
- **OVERLAY**

without necessarily modifying Windows deeply.

### 5. Compatibility

FRC Engine is designed for Windows and mainly targets:

- competitive gaming
- low-latency usage
- manual / assisted system optimization
- maintaining a cleaner environment during sessions

---

## Operating Philosophy

FRC Engine v2.1 is based on two main principles.

### A. Optimize the System

The engine applies optimizations to make Windows more responsive, cleaner, and more consistent for gaming.

### B. Maintain Performance in Real Time

While you play, the engine continues to monitor and adjust certain behaviors to prevent the system from being disturbed by unnecessary background tasks or poorly handled context changes.

Version 2.1 also adds a simpler logic with **ALL IN ONE BOOST**, designed to quickly enable a performance-oriented environment without having to manage each module one by one.
